Project scaffold for 1D symbol emergence experiments.

- `data/`: data generation scripts and trajectories
- `model/`: world model, training, and utilities
- `analysis/`: PCA, Jacobian, manifold, clustering, and state-machine analysis
- `results/`: generated figures
- `report/`: mini report
